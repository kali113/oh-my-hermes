# oh-worker

Use this when the user wants the agent to work through a concrete job end to end.

1. Create or update a work log with `oh-hermes secretary worklog`.
2. Break the request into a small execution checklist.
3. Use GBrain/MemOS for relevant user/project context before external research.
4. Execute with bounded tools, verify outputs, and record what changed.
5. Stop for explicit approval before irreversible, external, or account-affecting actions.
6. Convert repeated successful workflows into reusable skills or scripts.
