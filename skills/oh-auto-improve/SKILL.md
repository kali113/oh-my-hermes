# oh-auto-improve

Use this to improve the Hermes setup without uncontrolled drift.

1. Measure first: doctor, prompt-size, redaction, service health, and failing checks.
2. Generate candidate improvements in a report or branch.
3. Apply only low-risk local setup fixes automatically.
4. Never auto-merge prompt, skill, model, or memory-provider changes without review.
5. Keep before/after metrics and rollback commands.
